# Secure Multi-Client Encrypted Chat Application
A secure encrypted real-time messaging application developed using Python, Socket Programming, AES Encryption, and Tkinter GUI.

# Features 
* AES Encrypted Communication
* Multi-Client Chat System
* GUI Based Chat Application
* Real-Time Messaging
* TCP Socket Programming
* Secure Key Generation
* Message Logging

# Technologies Used
- Python
- Socket Programming
- Cryptography 
- Tkinter
- Kali Linux
